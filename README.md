# Application de gestion de stock en C#

Application console développée en C# permettant de gérer un stock de produits.

## Fonctionnalités

- Ajout de produits
- Suppression de produits
- Modification des quantités
- Affichage du stock

## Technologies utilisées

- C#
- Programmation orientée objet (POO)
- Visual Studio
